📄 Portfolio Project Description

Developed and deployed a personal portfolio website to showcase projects, technical skills, and professional links. The website was built as a responsive static web application using HTML, CSS, and JavaScript with a modern UI design.

The portfolio was hosted on Amazon S3 using static website hosting and configured with proper bucket policies for public access. A custom domain was purchased through Namecheap and integrated with the project to provide a professional web presence.

To ensure secure and fast delivery, AWS CloudFront was implemented as a Content Delivery Network (CDN). CloudFront was configured with an SSL certificate from AWS Certificate Manager to enable HTTPS and automatically redirect HTTP traffic to HTTPS, improving security and performance. DNS records were configured to connect the Namecheap domain with the CloudFront distribution.

This project demonstrates practical knowledge of cloud deployment, DNS configuration, CDN integration, and secure web hosting using AWS services.
